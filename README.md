# 📱 PRAIA - PhoneGap Build

## ¿Qué es Esta Carpeta?

Esta carpeta contiene el código listo para compilar en **PhoneGap Build**, que genera:
- ✅ **APK** (Android)
- ✅ **IPA** (iPhone/iPad)

Todo automáticamente sin necesidad de Mac, Android SDK, o configuración complicada.

## 🚀 Cómo Usar PhoneGap Build

### Paso 1: Crear Cuenta (Gratis)
```
1. Ve a: https://build.phonegap.com/
2. Presiona: "Sign Up"
3. Usa tu email: jota952009@gmail.com
4. Crea contraseña
5. Confirma tu email
```

### Paso 2: Conectar con GitHub (Opción A - Recomendada)

**Opción A1: Si tienes repositorio en GitHub**
```
1. Sube esta carpeta (PRAIA_PHONEGAP) a tu repositorio GitHub
2. En PhoneGap Build:
   - Presiona "New App"
   - Selecciona "Connect with GitHub"
   - Autoriza GitHub
   - Selecciona tu repositorio
   - Presiona "Create"
```

**Opción A2: Si NO tienes GitHub**
```
1. Crea cuenta gratuita en: https://github.com/signup
2. Crea repositorio nuevo: "PRAIA-Mobile"
3. Sube esta carpeta (PRAIA_PHONEGAP) al repositorio
4. Sigue Opción A1 desde el paso 2
```

### Paso 3: Cargar Directamente (Opción B - Sin GitHub)
```
1. En PhoneGap Build, presiona "New App"
2. Selecciona "Upload a .zip file"
3. Comprime esta carpeta (PRAIA_PHONEGAP) como ZIP
4. Sube el archivo ZIP
5. Presiona "Create"
```

### Paso 4: Compilar Android + iOS
```
1. En PhoneGap Build, ves tu proyecto
2. Presiona los botones para compilar:
   - Android: Presiona "Android"
   - iOS: Presiona "iOS"
3. Espera 2-5 minutos (PhoneGap compila en la nube)
4. Recibirás enlaces de descarga para:
   - APK (Android)
   - IPA (iPhone)
```

## 📥 Descargar los Archivos

Después de compilar en PhoneGap Build:

```
APK (Android):
- Descarga el archivo .apk
- Cópialo a tu teléfono Android
- Abre y presiona "Instalar"

IPA (iPhone):
- Descarga el archivo .ipa
- Usa iTunes o Xcode para instalar en iPhone
- O crea un cuenta Apple Dev para distribuir
```

## 🆘 Problemas Comunes

### "Error al compilar Android"
```
Solución:
1. Verifica que config.xml esté bien formado
2. Descarga nuevamente la carpeta
3. Recompila en PhoneGap Build
```

### "No puedo instalar IPA en iPhone"
```
Para instalar IPA en iPhone necesitas:
1. Cuenta Apple Developer (gratis)
2. Xcode instalado en Mac
3. O usar TestFlight para distribuir

Alternativa: Usar navegador web para PRAIA
```

### "PhoneGap Build dice que hay errores"
```
Pasos:
1. Descarga los logs desde PhoneGap Build
2. Lee los mensajes de error
3. Actualiza config.xml si es necesario
4. Recompila
```
