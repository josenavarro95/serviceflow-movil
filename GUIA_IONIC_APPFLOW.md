# 📱 Guía Rápida - Ionic AppFlow para PRAIA

## 🎯 ¿Qué es Ionic AppFlow?

Es un servicio similar a PhoneGap Build pero **más moderno, confiable y mantenido**:
- ✅ Compila Android (APK)
- ✅ Compila iOS (IPA)
- ✅ Automatizado en la nube
- ✅ Gratis para 1 app
- ✅ Servicio activo

---

## ⚡ En 10 Minutos (Paso a Paso)

### PASO 1: Crear Cuenta Ionic (3 min)
```
1. Abre: https://ionicframework.com/appflow
2. Presiona: "Sign Up" (esquina superior derecha)
3. Completa:
   Email: jota952009@gmail.com
   Contraseña: (la que tú quieras)
4. Verifica tu email
5. Inicia sesión
```

### PASO 2: Crear ZIP de tu Proyecto (2 min)
```
1. Abre: D:\INSTALADORES SERVICEFLOW\PRAIA_IONIC\
2. Busca: CREAR_ZIP.bat
3. Double-click
4. Espera a que termine
5. Se crea: PRAIA-ionic-appflow.zip en INSTALADORES SERVICEFLOW\
```

### PASO 3: Subir a Ionic AppFlow (3 min)
```
1. En Ionic Dashboard, presiona: "New App"
2. Selecciona: "Import from ZIP"
3. Sube: PRAIA-ionic-appflow.zip
4. Completa info:
   App Name: PRAIA
   App ID: com.praia.salon
5. Presiona: "Create App"
```

### PASO 4: Compilar (5 min)
```
1. En tu app PRAIA, presiona: "Build"
2. Selecciona:
   ✓ Android (APK)
   ✓ iOS (IPA)
3. Presiona: "Build"
4. Espera 5-10 minutos
```

### PASO 5: Descargar (2 min)
```
Cuando termine:
1. Presiona "Download" para APK
2. Presiona "Download" para IPA
3. Guarda ambos archivos
```

## 📱 Instalar en Dispositivos

### Android (APK)
```
1. Copia el APK al teléfono Android
2. Abre el archivo
3. Presiona: "Instalar"
4. Login: Cliente1 / 1234
5. ✅ Listo
```

### iPhone (IPA)
```
Opción A - Con Mac:
1. Abre Xcode en Mac
2. Conecta iPhone
3. Arrastra el IPA al iPhone
4. ✅ Listo

Opción B - Sin Mac (TestFlight):
1. Crea cuenta Apple Developer (gratis)
2. Sube IPA a TestFlight
3. En iPhone, descarga TestFlight app
4. Instala desde TestFlight
5. ✅ Listo

Opción C - Alternativa:
Usa navegador en iPhone para acceder a PRAIA
```

## 🔄 Actualizar la App

```
1. Modifica index.html
2. Ejecuta CREAR_ZIP.bat
3. En Ionic AppFlow, presiona "Update App"
4. Sube nuevo ZIP
5. Compila de nuevo
6. Descarga nuevos APK + IPA
7. Actualiza en tus dispositivos
```

## 🆘 Errores Comunes

### "Error en la compilación"
```
Soluciones:
1. Verifica que todos los archivos estén en PRAIA_IONIC/
2. Crea el ZIP de nuevo
3. Intenta subir nuevamente
```

### "No puedo instalar APK"
```
En Android:
1. Ve a Configuración → Seguridad
2. Activa: "Permitir apps desconocidas"
3. Intenta instalar de nuevo
```

### "No puedo instalar IPA en iPhone"
```
Sin Mac: Usa TestFlight (es más fácil)
Con Mac: Usa Xcode
```
